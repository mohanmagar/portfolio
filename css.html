CSS (Cascading Style Sheets)

Definition

My understanding is that CSS is a styling language used to control the visual presentation and layout of an HTML document. While HTML provides the raw structure and content, CSS brings it to life with colours, fonts, spacing, and responsive layouts. A key principle I've learned is that CSS allows for the separation of concerns, meaning the document's structure (HTML) is kept separate from its presentation (CSS), which makes the code much cleaner and easier to maintain.

Types of CSS Implementation

There are three main ways to add CSS to a webpage:

  • External Stylesheet: The CSS rules are written in a separate .css file and linked to the HTML document using a <link> tag in the <head>. This is the best practice because you can use one stylesheet to style an entire website, making it very efficient.
  • Internal Stylesheet: The CSS rules are placed inside a <style> tag directly within the <head> section of an HTML file. This is useful for styling a single, specific page.
  • Inline Styles: A CSS rule is applied directly to a single HTML element using the style attribute. This is generally avoided because it mixes content with presentation and is difficult to manage.

Working Example Explanation

My live example, css-example.css, is an external stylesheet that styles the html-example.html page. It demonstrates my understanding of several core CSS concepts to create a modern, responsive design:

  • Selectors: I have used a variety of CSS selectors to target specific elements, including element selectors (body, header), class selectors (.form-group), and pseudo-classes (:hover) to add interactivity to buttons and links.
  • The Box Model: I've applied padding and margin throughout the stylesheet to create space and improve readability. This demonstrates my understanding of how to control the space around and within elements.
  • Flexbox for Layout: To create a flexible and responsive layout, I have set the <main> element to display: flex. This allows the <article> and <section> elements inside it to align neatly and adapt to different screen sizes.
  • Media Queries for Responsive Design: I have included a media query (@media (max-width: 768px)) that changes the layout on smaller screens (like mobile phones). It switches the flex-direction to column, stacking the content vertically to ensure it remains easy to read and navigate on any device.

This stylesheet shows my ability to not only style a page but also to think about the user experience by making it responsive and well-structured.
