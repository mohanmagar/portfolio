Javascript

Definition

JavaScript is the programming language that makes websites interactive. If HTML provides the structure and CSS provides the style, JavaScript provides the functionality. It is a client-side language, meaning it runs in the user's browser, allowing me to create dynamic content, respond to user actions in real-time, and build features that make a website feel like an application.

How JavaScript is Written

From what I've learned in the module, JavaScript code is written as a series of statements. Key building blocks include:

  • Variables: These are like containers for storing data. I can declare a variable using let or const and give it a value, such as a number or a string of text.
  • Functions: A function is a reusable block of code that performs a specific task. I can define a function to do something, like generate a random number, and then "call" that function whenever I need it.
  • Control Flow: This includes things like loops (for loop) which allow my code to repeat an action multiple times.

JavaScript is typically written in a separate .js file and linked to an HTML document, which is the best practice for keeping code organized. But I have written in a HTML file by using script tag.

Working Example: A Random Color Generator

Live example

To demonstrate my skills, I have built a simple but visually effective Random Color Generator. This project clearly shows how JavaScript can respond to user input and manipulate the page's appearance in real-time.

Key Concepts Demonstrated:

  • Document Object Model (DOM) Manipulation: This is the most visible part of the project. My script directly manipulates the DOM in two ways:
    • Changing Styles: It changes the backgroundColor style property of the <body> element, instantly altering the entire page's background.
    • Updating Content: It updates the innerText of an <h2> element to display the new hexadecimal color code to the user.
  • Event Handling: The entire application is triggered by a user action.
    • Listening for Clicks: I've used addEventListener to listen for a click event on the main button. When the button is clicked, it executes the function responsible for changing the color.
  • Logic and Functions: The core logic is contained within a single, well-defined function.
    • Generating Random Values: The function contains a for loop that runs 6 times to build a random hexadecimal color code (e.g., #1A2B3C). It does this by randomly selecting characters from a predefined string of possible hex values. This demonstrates my ability to use loops and basic algorithms to solve a problem.

This project is a perfect example of how a few lines of JavaScript can add a fun and interactive feature to a webpage, combining event handling, logic, and direct manipulation of the page's content and style.
